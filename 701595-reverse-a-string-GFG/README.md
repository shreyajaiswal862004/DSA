# [Reverse a String](https://www.geeksforgeeks.org/problems/reverse-a-string/1)
## Easy
You are given a string s, and your task is to reverse the string.
Examples:
Input: s = "Geeks"
Output: "skeeG"

Input: s = "for"
Output: "rof"
Input: s = "a"
Output: "a"
Constraints:1 &lt;= s.size() &lt;= 106s contains only alphabetic characters (both uppercase and lowercase).
