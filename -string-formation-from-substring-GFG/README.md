# [String formation from substring](https://www.geeksforgeeks.org/problems/string-formation-from-substring2734/1?utm=codolio)
## Medium
Given a string s, determine whether it can be formed by repeating one of its substrings multiple times.
Examples:
Input: s = "geeksgeeksgeeks"
Output: true
Explanation: It is constructed by repeating "geeks" 3 times. So answer for this test case is true.
Input: s = "geeksforgeeks"
Output: false
Explanation: No substring can be repeated multiple times to form "geeksforgeeks". So answer for this test case is false.
Constraints:1 ≤ n ≤ 105, n is length of string s.string s contains only lowercase english letter.
