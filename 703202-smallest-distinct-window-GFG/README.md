# [Smallest distinct window](https://www.geeksforgeeks.org/problems/smallest-distant-window3132/1?utm=codolio)
## Medium
Given a string&nbsp;str,&nbsp;your task is to find the&nbsp;length&nbsp;of the&nbsp;smallest&nbsp;window that contains&nbsp;all&nbsp;the characters of the given string at least once.
Example:

Input: str = "aabcbcdbca"
Output: 4
Explanation: Sub-String "dbca" has the smallest length that contains all the characters of str.


Input: str = "aaab"
Output: 2
Explanation: Sub-String "ab" has the smallest length that contains all the characters of str.
Input: str = "geeksforgeeks"
Output: 7
Explanation: There are multiple substring with smallest length that contains all characters of str, "eksforg" and "ksforge". 
Constraints:1 ≤ str.size() ≤ 105str contains only lower-case english alphabets.
